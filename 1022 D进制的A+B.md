## 1022 D进制的A+B （20 分）

输入两个非负 10 进制整数 A 和 B (≤2​^30^−1)，输出 A+B 的 D (1<D≤10)进制数。

**输入格式：**
输入在一行中依次给出 3 个整数 A、B 和 D。

**输出格式：**
输出 A+B 的 D 进制数。

**输入样例：**

    123 456 8

**输出样例：**

    1103

**代码**

```c
#include <cstdio>
int main () {
	int a, b, d ;
	scanf ("%d %d %d", &a, &b, &d) ;
	int sum = a+b ;
	int ans[100] ;
	int n=0;
	do {						//十进制到d进制转化使用do while 语句 
		ans[n++] = sum % d ;	//取余 
		sum /= d ;				//取商 
	} while (sum!=0) ;
	for (int i=n-1; i>=0; i--) {//从高位到低位输出 n-1 ~ 0 
		printf ("%d", ans[i]) ;
	}	
	return 0 ;
}
```
